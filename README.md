### 调试日志：

#### 2026.1.5

已经实现行径轮的力控底盘控制，关节电机目前只能开环控制，即让关节转到指定角度。实际IMU闭环正在做，不过还没想好该怎样实现闭环控制。对于DM的MIT或者位置速度模式都需要输入目标角度和速度值，目前困惑点在于：我知道了现在电机PITCH轴的倾斜角度，这个角度该如何赋给电机，因为车体倾斜角度和腿伸长量有关，腿伸长量又和关节电机转动角度有关，但是关节电机的转动角度和车体倾斜角也不是线性对应的关系，似乎只跑P控制器赋值给电机没有什么意义，暂时还没想通这个。

调试时注意关节不能长时间维持抬起状态，否则电机持续升温，最终过温就会“死”掉。
